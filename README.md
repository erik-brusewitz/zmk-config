----- Keyboard components -----
Case: Manta58s + short tenting legs
Screen: nice!view 
Microcontroller: nice nano v2 
PCB: aurora lily58 v2
Switches: Akko penguin



----- Layered bindings -----

----- Menu = Lower -----
J - home
K - pgdown
I - pgup
L - end
{} - ()

----- Func = Raise -----
J - left
K - down
I - up
L - right
{} - []

----- Both layers -----
Esc  - BTCLR
Tab  - BT1
Q    - BT2
1-0  - F1-F10.
+    - F11
F1   - F12
<>   - ,.
T    - ~ (Tilde)
A    - * (Asterisk)
S    - @ (Snabela)
D    - $ (Dollar)
F    - ' (appostroF)
G    - ´ (Grave (Normal accent, and layer+shift+G is grave accent).)
C    - ^ (Caret)
V    - | (Vertical bar)
B    - \ (Backslash)
P    - § (Paragraf)
